API Documentation

Users

Update User Info

Method: PUT /users/:id

Description: Updates the user information (username, password, email, date of birth).

Request Body:

{
  "username": "new_username",
  "password": "new_password",
  "email": "new_email@example.com",
  "dateOfBirth": "1990-01-01"
}

Response:

{
  "message": "User information updated successfully",
  "user": {
    "_id": "66d5f2fac4e74527382710c8",
    "username": "new_username",
    "email": "new_email@example.com",
    "firstName": "Peter",
    "lastName": "Parker",
    "birthday": "1995-08-10T00:00:00.000Z",
    "favoriteMovies": []
  }
}

Add Favorite Movie

Method: POST /users/:id/favorites/:movieId

Description: Adds a movie to the user's list of favorites.

Response:

{
  "message": "Movie added to favorites",
  "user": {
    "_id": "66d5f2fac4e74527382710c8",
    "favoriteMovies": [
      "66d5f344c4e74527382710cb"
    ]
  }
}

Remove Favorite Movie

Method: DELETE /users/:id/favorites/:movieId

Description: Removes a movie from the user's list of favorites.

Response:

{
  "message": "Movie removed from favorites",
  "user": {
    "_id": "66d5f2fac4e74527382710c8",
    "favoriteMovies": []
  }
}

Deregister User

Method: DELETE /users/:id

Description: Deletes the user account.

Response:

{
  "message": "User deregistered successfully"
}
